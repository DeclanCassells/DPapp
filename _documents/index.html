How to disable tocuh swipe function without disable Menu ?

You can disable touch swipe function without disable menu on assets > js > custom.js Find code below and delete that.; 

// Delete Code below for stop swipe left & right function.
		
// Swipe left function for menu. If you want you can disable swipe left funtion delete code below.
$('#menu, .pages').live("swipeleft", function(){
    if (menuStatus){	
    $(".ui-page-active").animate({
        marginLeft: "0px",
      }, 0, function(){menuStatus = false});
      }
});
// Swipe left end


// Swipe right function for menu. If you want you can disable swipe right funtion delete code below.
$('.pages').live("swiperight", function(){
    if (!menuStatus){	
    $(".ui-page-active").animate({
        marginLeft: "165px",
      }, 0, function(){menuStatus = true});
      }
});
// Swipe right end

//End Delete Code

Home Page Slider

You can change slider transition effects on this section. And also youcan find example files on main menu HOME section > index.html. The slider can be change horizontal or vertical and fade or slide effects. Example code is on slider page header section code below:
For Vertical Slide: You can change it "horizontal" or "vertical"


$(document).ready(function() {
	$('.flexslider').flexslider({
		slideDirection: "vertical",
        start: function(slider){
				$('.flexslider').css("max-height" , "574px");
	  });
});

For fade or sliding slide: You can chane if "fade" or "slide"


$(document).ready(function() {
	$('.flexslider').flexslider({
		slideDirection: "horizontal",
		animation: "fade",
        start: function(slider){
				$('.flexslider').css("max-height" , "574px");
	  });
});

and also you can edit default parameter for Slider on assets>js > jquery.flexslider.js codes below:

Add Home Bubble

The homepage uses a script which creates a "Add to Home" bubble on iPhones and iPads. This reminds people that they can install an icon on their device which will create a web app on their device. To edit the icon used in the bubble open and edit assets > images > touch

To change settings of the bubble open assets > js > add2home.js You can change the animation entering and leaving the page, how long you want the bubble to stay on the page and how long before it shows up again.

options = {
autostart: true,			// Automatically open the balloon
returningVisitor: false,	// Show the balloon to returning visitors only (setting this to true is HIGHLY RECCOMENDED)
animationIn: 'drop',		// drop || bubble || fade
animationOut: 'fade',		// drop || bubble || fade
startDelay: 2000,			// 2 seconds from page load before the balloon appears
lifespan: 15000,			// 15 seconds before it is automatically destroyed
bottomOffset: 14,			// Distance of the balloon from bottom
expire: 0,					// Minutes to wait before showing the popup again (0 = always displayed)
message: '',				// Customize your message or force a language ('' = automatic)
touchIcon: false,			// Display the touch icon
arrow: true,				// Display the balloon arrow
hookOnLoad: true,			// Should we hook to onload event? (really advanced usage)
iterations: 100				// Internal/debug use

Integrate with Site

HTML Site

If you already have a HTML web site or ASP, PHP etc.. you can easily integrate with your exiting web site. Just use our Mobile Detect JS ( code is on left menu ) and put it on your index.html or default.asp or index.php file's head section like i tell on Mobile Detect Code. And create a folder called mobile on your root directory. Copy all files from theme zipped document to this folder. And configure file and folder names on mobile detect JS.

That is it you have done!

Wordpress Site

If you already have a Wordpress web site you can easily integrate with your exiting web site. Just use our Mobile Detect JS ( code is on left menu ) and put it on your default wordpress's template header.php file's head section like i tell on Mobile Detect Code. And create a folder called mobile on your template's root directory. Copy all files from theme zipped document to this folder. And configure file and folder names on mobile detect JS.

That is it you have done!
